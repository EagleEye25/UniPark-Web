User Information
Will display all of your information.

Update User Information
This allows you to update either your: Cell-Phone Number, E-mail Address, Password.
Cell-Phone Number: Needs to have 10 digits entered as specified.
E-mail Address: Only valid email addresses will be accepted.
Password: When entering a new password, it will need to be entered again inorder to ensure that you entered the same password.
By pressing the or icon it will display, hide your newly entered password.

Assigned Parking
Will display all parking information associated to you.

Request Parking This allows you to request for a different parking area / spot within the facility.
