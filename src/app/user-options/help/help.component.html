Update User Information
This allows you to update either your: Cell-Phone Number, E-mail Address, Password.
Cell-Phone Number: Needs to have 10 digits entered as specified.
E-mail Address: Only valid email addresses will be accepted.
Password: When entering a new password, it will need to be entered again inorder to ensure that you entered the same password.
By pressing the or icon it will display, hide your newly entered password.

Assigned Parking
Will display all parking information associated to you.

Request Parking
This allows you to request for a different parking area / spot within the facility.
1) First select the desired parking area.
2) Then select a parking spot.
3) Finally click the request button.

View Requests
This will display all your currently (In progress) requests.
A request can be canceled by:
1) Clicking the Cancel button, a confirmation box will appear asking if you want to cancel.
    *Clicking NO will not cancel the request.
    *Clicking YES will cancel the request, and the request will be removed.
