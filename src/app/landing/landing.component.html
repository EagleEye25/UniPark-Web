Dragon Code is a group of third year students from Mandela University, that have been given the task to create an "Advanced", secure parking system for the university. In which they saw greater opertunities, and decided to develope their project, so that it could be used by not only the university, but any other minor, major facilities that wish to ensure that their personnel have save parking areas, with little to no chance of theft of vehicles, and also improve the way in which they monitor and control who has access to the facility, and when.

UniPark is an open souce project for a fully automated parking system, that will allow users that have access to a facilities parking area to enter. The program is securely setup to prevent unauthorized users access to the facility, also to help the facility to control traffic, ensure that person/s in the facility park in the correct area, which is assigned to them. The aim of the system is to improve security of the facility, and ensure that little to no theft occurs regarding motor vehicles.
